3.7 G

3.7.1 GUID Alarm Code


GUID-000 WARN Speed limit exceeded

Cause: Command speed exceeded limits.

Remedy: Reduce sensor gain.


GUID-001 WARN Rotation speed limit exceeded

Cause: Command rotation speed exceeded limits.

Remedy: Reduce sensor gain.


GUID-002 SERVO Excessive motion command

Cause: Motion command exceeded limits.

Remedy: Reduce sensor gain.


GUID-003 SERVO Teach Device Enabled

Cause: Other source tried to execute motion while LeadThru device was enabled.

Remedy: Disable teach device.


GUID-004 WARN Limit error, axis num %d^3

Cause: Limit or singularity encountered.

Remedy: Move in other direction.


GUID-005 WARN Torque limit, axis num %d^3

Cause: Motor torque disturbance detected.

Remedy: Move away from object if collision occurred.


GUID-006 SERVO Motion limit exceeded

Cause: Software motion limit exceeded.

Remedy: Check SMON for details.


GUID-007 SERVO Excess command torque %d^3

Cause: Command torque exceeded set limits.

Remedy: Reduce gains or velocity limit parameters to avoid exceeding limits.


GUID-008 SERVO Excess joint velocity %d^3

Cause: Joint velocity exceeded set limits.

Remedy: Reduce gains or velocity limit parameters to avoid exceeding limits.


GUID-009 WARN Joint velocity limit %d^3

Cause: Joint velocity exceeded set limits.

Remedy: Reduce gains or velocity limit parameters to avoid exceeding limits.


GUID-010 SERVO Soft OVC alarm (A:%d^3)

Cause: Automatic Motion Disabled A program attempted motion commands while the IAD is in manual control mode.

Remedy: Manual mode must be disabled before automatic motion is permitted.


GUID-050 ABORT Automatic Motion Disabled

Cause: Automatic Motion Disabled A program attempted motion commands while the IAD is in manual control mode.

Remedy: Manual mode must be disabled before automatic motion is permitted.


GUID-051 WARN Liveman switch tied down

Cause: Liveman switch tied down or liveman hardware failure.

Remedy: Remove interfering object, increase TIE_TIMER, reduce TIE_THRES, or repair liveman hardware.


GUID-052 WARN Tracking fault %d^3

Cause: An unexpected fault occurred during line tracking.

Remedy: Follow instructions for indicated fault number.


GUID-053 WARN Excessive Track Speed. L:%d^3

Cause: The line speed indicated by the encoder is too fast.

Remedy: Fix encoder problem or reduce line speed to safe level.


GUID-054 WARN End of Travel Warning. L:%d^3

Cause: The operator crossed the end of travel warning point while line tracking.

Remedy: The line speed may be too fast to finish the cycle in time.


GUID-055 WARN End of Travel Hold. L:%d^3

Cause: The operator crossed the end of travel hold point while line tracking.

Remedy: The line speed may be too fast to finish the cycle in time.


GUID-056 WARN End of Travel Stop. L:%d^3

Cause: The operator crossed the end of travel stop point while line tracking.

Remedy: The line speed may be too fast to finish the cycle in time.


GUID-057 WARN Can’t init tracking. L:%d^3

Cause: Tracking won’t start due to an initialization problem. Check SMON for details.

Remedy: Fix problem indicated on SMON.


GUID-058 WARN Queue overflow. L:%d^3

Cause: Tracking queue overflowed: too many parts in the queue. The oldest queue entry was lost.

Remedy: Set LEN, OUT_BOUNDS, and DETECT_DIST to reasonable values. Relocate part detect switch closer to work area.


GUID-059 SERVO Current queue lost. L:%d^3

Cause: Lost queue entry was in use.

Remedy: Set LEN, OUT_BOUNDS, and DETECT_DIST to reasonable values. Relocate part detect switch closer to work area.


GUID-060 WARN LT File R/W while moving.

Cause: LT File R/W while moving. Attempted to read or write logic table text file while the IAD is in operation.

Remedy: Engage brakes before logic table file read or write.


GUID-100 WARN Unknown error

Cause: An internal error has occured.

Remedy: Cold start the controller


GUID-101 WARN Cannot run monitoring task

Cause: Cannot run monitoring task Cannot start the KAREL shell program to monitor I/O events

Remedy: Abort unwanted/paused KAREL programs. Or increase maximum number of tasks at control start.


GUID-102 WARN Cannot calibrate sensor.

Cause: Cannot calibrate sensor. Cannot calibrate the force sensor.

Remedy: No lead thru teaching motion is allowed when calibrating the force sensor.


GUID-103 WARN TPE operation error.

Cause: TPE operation error. This error occures when lead thru teach cannot access the TPE program.

Remedy: Make sure the TPE program exists.


GUID-104 WARN Invalid TPE program name.

Cause: Invalid TPE program name. The TPE program does not exist.

Remedy: Make sure the TPE program exists.


GUID-105 WARN Robot is not ready.

Cause: Robot is not ready. Cannot move or record a

Remedy: Make sure the robot is mastered and no other errors.


GUID-106 WARN Lead Thru is not loaded.

Cause: Lead Thru not loaded. The lead thru teach software does not load properly.

Remedy: Reload the lead thru software


GUID-107 WARN Invalid Single Step Mode.

Cause: Invalid Single Step Mode. The lead thru teach software does not work in single step mode.

Remedy: Turn off the single step mode


GUID-108 WARN No data read from AIN port

Cause: No data read from AIN port No data comes from the AIN ports.

Remedy: Check AIN port connection, AIN port module and AIN port setting.


GUID-150 WARN Analog ports are invalid.

Cause: Analog ports are invalid. The analog port setting is invalid.

Remedy: Please check the DIO menu for for proper AIN setting.


GUID-151 WARN Unusual force sensor load.

Cause: Unusual force sensor load. New force sensor zero setting exceeded change limits during auto-zeroing.

Remedy: Remove foreign object from operator controls or increase RESET_LIMIT, press FS Reset button on controller.


GUID-152 SERVO Force sensor overload A:%d^3

Cause: Force sensor overload A:%d^3 Force measurement exceeded allowed limits.

Remedy: Reduce force on force sensor.


GUID-153 SERVO Force sensor ROM error

Cause: Force sensor ROM error Copying a DSP program from ROM to RAM failed (DSP init).

Remedy: Replace sensor head, interface board, and/or cable.


GUID-154 SERVO Force sensor SLC-2 init

Cause: Force sensor SLC-2 init SLC-2 (Serial Link Controller-2) initialization failed (DSP init).

Remedy: Replace sensor head, interface board, and/or cable.


GUID-155 SERVO Force sensor SLC-2 init

Cause: Force sensor SLC-2 init SLC-2 (Serial Link Controller-2) initialization failed (DSP init).

Remedy: Replace sensor head, interface board, and/or cable.


GUID-156 SERVO Force sensor data req fail

Cause: Force sensor data req fail Data request to sensor head failed.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-157 SERVO Force sensor outer parity

Cause: Force sensor outer parity Parity error on FS I/F board outer RAM.

Remedy: Replace interface board.


GUID-158 SERVO Force sensor inner parity

Cause: Force sensor inner parity Parity error on FS I/F board inner RAM.

Remedy: Replace interface board.


GUID-159 SERVO Force sensor comm failure

Cause: Force sensor comm failure Communication failed on FS I/F board.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-160 SERVO Force sensor comm failure

Cause: Force sensor comm failure Communication failed on FS I/F board.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-161 SERVO Force sensor comm failure

Cause: Force sensor comm failure Communication failed on FS I/F board. (parity, framing, CRC, etc.)

Remedy: Replace sensor head, interface board, and/or cable.


GUID-162 SERVO F-bus system emergency

Cause: F-bus system emergency Non-force sensor system emergency error on F-bus

Remedy: Correct source of error.


GUID-163 SERVO Force sensor watch dog

Cause: Force sensor watch dog Force sensor interface watchdog alarm.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-164 SERVO Force sensor shared parity

Cause: Force sensor shared parity Parity error in shared memory.

Remedy: Replace interface board.


GUID-165 SERVO Force sensor local parity

Cause: Force sensor local parity Parity error in local memory.

Remedy: Replace interface board.


GUID-166 SERVO Force sensor SIR1 SLC-2

Cause: Force sensor SIR1 SLC-2 SLC-2, SLDM, or SLDS error.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-167 SERVO Force sensor dist link comm

Cause: Force sensor dist link comm Distributed link error between head and interface.

Remedy: Replace sensor head, interface board, and/or cable.


GUID-200 WARN TCP Mate motion aborted.

Cause: TCP Mate motion aborted. The TCP Mate motion aborted due to unexpected change of of menu, or aborted by the user.

Remedy: Please start over.


GUID-201 WARN Form Operation Error.

Cause: Form Operation Error. Dictionary not found.

Remedy: Please reload the controller.


GUID-202 WARN System is not initalized.

Cause: System is not initalized. System not been fully initialized.

Remedy: Use the Setup menu to set up all required variables.


GUID-203 WARN Fail to run TCPMate Motion

Cause: Fail to run TCPMate Motion Cannot start the TCP Mate motion.

Remedy: Clear all the errors before running TCP Mate motion.


GUID-204 WARN Please record all position

Cause: Please record all position Not all positions in the reference menu are recorded.

Remedy: All positions in the menu should have been recorded.


GUID-205 WARN Do reference motion first.

Cause: Do reference motion first. Cannot execute the TCP mate without mastering TCP first.

Remedy: Please go to reference menu and press MASTER to run reference motion first.


GUID-206 WARN Point is not initialized.

Cause: Point is not initialized. Position is not initialized.

Remedy: Please record positions in the reference menu first.


GUID-207 WARN Robot is not calibrated.

Cause: Robot is not calibrated. Cannot record position since the robot is not calibrated

Remedy: Calibrate the robot first.


GUID-208 WARN Record home position first

Cause: Record home position first Please record approach point before record other positions

Remedy: Record home position.


GUID-209 WARN Keep same WPR as home pos.

Cause: Keep same WPR as home pos. These position require to have the same WPR as the home position.

Remedy: Keep the same orientation when recording these points.


GUID-210 WARN Please press SHIFT key.

Cause: Please press SHIFT key. Press SHIFT key at the same time when this key is pressed

Remedy: Press SHIFT key to active the action.


GUID-211 STOP TCP differs by %s

Cause: TCP differs by %s The current TCP and previous differs more than pre-defined threshold.

Remedy: Check for loose hareware or too. Run TCPMate again to confirm the TCP value.


GUID-212 WARN XYZ: %s

Cause: XYZ: %s TCPMate logs the previous TCP XYZ value to error log for record keeping.

Remedy: None.


GUID-213 WARN WPR: %s

Cause: TCPMate logs the previous TCP WPR value to error log for record keeping.

Remedy: None.


GUID-214 WARN Position is not in range.

Cause: Position is not in range. The destination position is not reachable.

Remedy: Re-teach a new starting pos.


GUID-215 WARN Singular solution.

Cause: Singular solution. Cannot compute the circle due to singularity.

Remedy: Check and correct robot’s mastering. Check hardware. Try reteach positino on the Reference menu.


GUID-216 WARN Cannot converge.

Cause: Cannot converge. Cannot converage to a a solution.

Remedy: Check and correct robot’s mastering. Check hardware. Try reteach positino on the Reference menu.


GUID-217 WARN Points are too close.

Cause: Points are too close. Recorded positions are too close to each other.

Remedy: Points should be at least 10 mm apart.


GUID-218 WARN Contact before search.

Cause: Contact before search. Robot is in contact with the part before starting a search motion.

Remedy: Reteach the starting position


GUID-219 WARN No contact detected.

Cause: No contact detected. No contact is made druing the TCP Mate motionl.

Remedy: Make sure the sensor and software are installed correctly.


GUID-220 WARN Undefined program name.

Cause: Undefined program name. The enter/exit program names are not defined.

Remedy: Go to the setup menu to select TPE programs.


GUID-221 WARN TPE operation error.

Cause: TPE operation error. Internal TPE program error.

Remedy: Please reload the controller.


GUID-222 WARN Circle Fitting error: %s.

Cause: Circle Fitting error: %s. The circle fitting error is greater than expected.

Remedy: Check hareware/tooling or increase fit tolerance.


GUID-223 WARN No DIN support.

Cause: No DIN support. The search motion can not be executed withtout TOS sensor.

Remedy: Go to setup menu and select TOS in Sensor Type.


GUID-224 WARN Cannot continue motion.

Cause: Cannot continue motion. Cannot continue the paused motion.

Remedy: Abort the paused motion and re-run TCP Mate again.


GUID-225 WARN TCP Mate internal error.

Cause: TCP Mate internal error. TCP Mate internal error.

Remedy: Power off to recover. If problem persistes, please reload the controller.


GUID-226 WARN Dictionary is not loaded.

Cause: Dictionary is not loaded. Dictionary file is not loaded

Remedy: Please reload the controller.


GUID-227 WARN No saved reference data.

Cause: No saved reference data. No saved reference data for current tool.

Remedy: Please do reference motion for the current tool.


GUID-228 WARN File open error: %s

Cause: File open error: %s Cannot open output file.

Remedy: Check the path and file name.


GUID-229 WARN No recorded Utool.

Cause: No recorded Utool. No recorded utool to save.

Remedy: Recorded a new utool.


GUID-230 WARN Not enough TPE stack.

Cause: Not enough TPE stack. The TPE program does not have enough stack size to run TCPMate automatically.

Remedy: Go to the TCPMATE main menu, TCPMate will update your TPE program stack size once you are in the menu. Leave the menu, re-run the TPE program again.


GUID-231 ABORT Cannot generate TCP offset

Cause: Cannot generate TCP offset TCPMate cannot complete it’s operation successfully.

Remedy: Check for sever tool bent or loosen hardwares.


GUID-232 STOP Collision, Axis %d^3 (G:$d^2)

Cause: Collision (G:$d^2, A:%x^3 Hex) Collision during motion

Remedy: Reteach starting position or remove collision object.


GUID-250 WARN Software loading error.

Cause: Software loading error. Software loading error.

Remedy: Please reload the controller.


GUID-251 WARN Invalid data range.

Cause: Invalid data range. The specified data range is invalid.

Remedy: Please check the max_value and min_value in under the setup.


GUID-252 WARN Invalid I/O Port setting.

Cause: Invalid I/O Port setting. Cannot access to the specified I/O port.

Remedy: Please use the I/O menu to check the port setting.


GUID-253 WARN Invalid system variable.

Cause: Invalid system variable. Cannot access to the specified system variable name.

Remedy: Check the spelling of the system variable.


GUID-254 WARN Invalid torque setup.

Cause: Invalid torque setup. Cannot access to specified torque data.

Remedy: Please reload the controller.


GUID-300 WARN No contact with the chuck.

Cause: No contact with the chuck. Robot does not have a contact with the chuck during search motion.

Remedy: Increase the search distance under the setup menu, or move the taught position close to the chuck.


GUID-301 WARN Cannot update prostion reg

Cause: Cannot update prostion reg Cannot update the specific position register.

Remedy: Make sure the position register number is valid.


GUID-302 WARN Failed to run search motn.

Cause: Failed to run search motn. Cannot start the motion task.

Remedy: Aborted paused KAREL tasks in the controller.


GUID-303 WARN Other task is running.

Cause: Other task is running. Another motion task is running cannot start a new motion.

Remedy: Aborted the running task before executing a search motion.


GUID-350 WARN Invalid SoftWall: %s

Remedy: Move the cursor to the ARC start line, then press the Simple Teach execution function key.
